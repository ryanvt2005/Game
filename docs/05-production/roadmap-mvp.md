# MVP Roadmap (Act I Vertical Slice)

## Milestone 1 — Prototype Feel
- Third-person camera (follow + collision)
- Movement: run/jump/dash
- Basic attack + hit reactions
- One test arena

## Milestone 2 — One Hero Complete
- Ability system (cooldowns + energy)
- 3 abilities + ultimate (first hero)
- Basic UI HUD

## Milestone 3 — Enemies + Boss
- Drone enemies (AI + waves)
- One villain boss
- One corrupted hero boss (multi-phase)

## Milestone 4 — Act I Missions
- 6–10 missions in one district
- Objectives + triggers + dialogue stubs
- Reputation tracking

## Milestone 5 — Polish + Performance
- VFX + SFX pass
- Load times + asset budgets
- Playable demo build
