# Fracture: Ascension (Working Title)

A 3D single-player superhero action game playable in the browser.

## Current Direction
- Mode: Single-player story campaign first
- Setting: Modern city zones impacted by "The Fracture" (unstable power event)
- Conflict: Player fights villains and heroes (ideology + power instability)

## Repo Layout
- `/docs` — design, narrative, technical documentation
- `/.github/ISSUE_TEMPLATE` — structured issues for agentic coding tools
- `/src` — application + game code (to be added)
